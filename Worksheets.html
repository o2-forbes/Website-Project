My Wireframe contains my initial plan for the creation and design of my website template. It includes reference labels and sketches which therefore helped me progress in the production phase of this task.

A wireframe is intended to plan the production of a website and consists of a visual representation along with an outline that depicts the basic structure and layout of the web page. It is a simplified design that focuses on the placement of elements, such as navigation, content, and functionality, without incorporating detailed design elements like colors, images, or specific fonts.

The body elements being the foundation of your HTML file that will be expanded on in the CSS file that we later learnt to link.
