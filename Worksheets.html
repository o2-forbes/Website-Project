My Wireframe contains my initial plan for the creation and design of my website template. It includes reference labels and sketches which therefore helped me progress in the production phase of this task.

A wireframe for a website is a visual representation or a skeletal outline that depicts the basic structure and layout of a web page. It is a low-fidelity, simplified design that focuses on the placement of elements, such as navigation, content, and functionality, without incorporating detailed design elements like colors, images, or specific fonts.
